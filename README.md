# CodeAlpha_Unemployment_Analysis
A comprehensive Data Science project analyzing the impact of COVID-19 on India's unemployment rates. Features data cleaning, time-series visualization, and regional comparisons (Rural vs. Urban) using Python, Pandas, and Seaborn. Completed for the CodeAlpha Internship Task 2.

# Unemployment Analysis with Python
This project is part of my internship at CodeAlpha.

## Objective
To analyze the unemployment rate in India, specifically focusing on the impact of the COVID-19 pandemic across different states and regions (Rural vs Urban).

## Libraries Used
* Pandas
* Matplotlib
* Seaborn
* Plotly
